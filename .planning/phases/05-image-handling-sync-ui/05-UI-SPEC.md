## Visual Hierarchy

**Focal point:** The green sync button (`bg-green-600`) in the Sync Trigger Card is the primary visual anchor — draws the eye first due to its color contrast against the dark gray background.

**Hierarchy order:**
1. Sync button (green CTA) — primary action, most prominent
2. Page heading "CSDN 同步" — establishes context
3. Config form card — required setup before sync is useful
4. Sidebar nav "CSDN 同步" item — entry point to page

## Component Inventory

### Admin Sidebar (AdminSidebar.vue)
- Extends existing pattern with new nav item
- New item: router-link to "/admin/csdn-sync" with active state matching existing pattern
- Active state: `bg-blue-600 text-white`
- Default state: `text-gray-400 hover:bg-gray-800 hover:text-gray-100`
- Icon: none (matches existing nav items)

### Sync Config Card
- Wrapper: `bg-gray-800 rounded-lg p-6 border border-gray-700 mb-6`
- Heading: `text-lg font-semibold text-gray-100 mb-4`
- Grid: `grid grid-cols-2 gap-4`
- Input: `w-full px-3 py-2 bg-gray-700 border border-gray-600 rounded text-gray-100`
- Select: same as input
- Label: `block text-sm text-gray-400 mb-1`
- Save button: `mt-4 px-4 py-2 bg-blue-600 hover:bg-blue-700 text-white rounded-lg`

### Sync Trigger Card
- Wrapper: `bg-gray-800 rounded-lg p-6 border border-gray-700`
- Sync button: `px-6 py-3 bg-green-600 hover:bg-green-700 text-white rounded-lg font-semibold`
- Sync button disabled: `disabled:opacity-50`
- Loading spinner: text "同步中..." replaces button text

### Sync Results Panel (D-01 inline panel)
- Appears below sync button when `syncResult` is not null
- Wrapper: `mt-4 p-4 bg-gray-700 rounded-lg border border-gray-600`
- Results text: `text-gray-300`
- Success count: `text-green-400 font-bold` with `{{ syncResult.created }}`
- Update count: `text-blue-400 font-bold` with `{{ syncResult.updated }}`
- Skip count: `text-yellow-400 font-bold` with `{{ syncResult.skipped }}`
- Error count: `text-red-400 font-bold` with `{{ syncResult.errors?.length || 0 }}`
- Error list: `mt-2 text-red-400 text-sm` with `v-for` error items

### Sync Error Banner
- Shown when `syncError` is not empty
- Wrapper: `mt-4 p-4 bg-red-900 bg-opacity-30 border border-red-700 rounded text-red-400`
- Text: {{ syncError }}

### Synced Article Warning Modal (D-03 blocking modal)
- Overlay: `fixed inset-0 bg-black bg-opacity-50 flex items-center justify-center z-50`
- Card: `bg-gray-800 rounded-lg p-6 w-full max-w-md border border-gray-700`
- Heading: `text-lg font-semibold text-yellow-400 mb-4` with text "编辑警告"
- Body: `text-gray-300 mb-6`
- CTA button: `w-full px-4 py-2 bg-yellow-600 hover:bg-yellow-700 text-white rounded-lg`
- Button text: "我已知悉"
- Blocking: no other actions available until dismissed

### CsdnSyncView Layout
- Page wrapper: `flex min-h-screen bg-gray-900`
- Sidebar: `<AdminSidebar />`
- Main content: `flex-1 p-8`
- Page heading: `text-3xl font-bold text-gray-100 mb-8`

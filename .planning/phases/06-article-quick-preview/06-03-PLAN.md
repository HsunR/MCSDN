### Task 2: Import Modal and Add State

<read_first>
frontend/src/views/public/HomeView.vue
</read_first>

<action>
修改 `frontend/src/views/public/HomeView.vue`：

**1. 添加导入：**
```javascript
import ArticlePreviewModal from '../../components/public/ArticlePreviewModal.vue'
```

**2. 添加 state：**
```javascript
const previewArticleId = ref(null)

function handlePreview(articleId) {
  previewArticleId.value = articleId
}

function handleClosePreview() {
  previewArticleId.value = null
}
```

**3. 绑定到 ArticleTimeline：**
```vue
<ArticleTimeline
  :articles="store.articles"
  @preview="handlePreview"
/>
```

**4. 添加 Modal 到 template：**
```vue
<ArticlePreviewModal
  :visible="previewArticleId !== null"
  :articleId="previewArticleId"
  @close="handleClosePreview"
/>
```
</action>
